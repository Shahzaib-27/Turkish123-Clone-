# Turkish123-Design
Turkish123 CLone
<br>
This project is a clone of the Turkish123 website, developed using HTML, CSS, and JavaScript. It features a clean and responsive layout that showcases Turkish dramas and series with thumbnails, categories, and a search bar. The design replicates the look and feel of the original site, offering users an engaging browsing experience across all devices.
